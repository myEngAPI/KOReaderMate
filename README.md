## 📚 KOReaderMate


KOReader的生词和标注导出软件，可用导出任意格式的生词或标注，还可用通过插件进行WebDAV导出。



## 📥  支持平台

 **Windows**
 
 **Linux**

## 🔧使用方法

**WebDAV**

  注册支持WebDAV的服务商，如坚果云
  
  获得WebDAV的用户名密码
  
**生词同步**

  KOreader顶上第五个图标放大镜 🔍-词汇本-同步
  
**标注，摘录同步**

  安装插件：https://github.com/gitalexcampos/koreader-Highlight-Sync
